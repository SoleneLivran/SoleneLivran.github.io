Lily's Library

Projet personnel
Hapi.js · Nuxt.js

Projet en cours de développement

Interface développée à destination de mes amis, afin de partager avec eux le contenu de ma bibliothèque, de demander un emprunt, et de suivre les disponibilités.

Toutes les informations sur les livres ont été compilées via leur n° ISBN ou des ajouts manuels via l'application "Ma bibliothèque", à partir de laquelle est exporté un fichier csv contenant toutes les données nécessaires.
